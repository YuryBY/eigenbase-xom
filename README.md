# eigenbase-xom

XML object model for Java.

# Prerequisites

Eigenbase-XOM requires git, maven (3.0.4 or later), and JDK 1.6 or later (JDK 1.7 preferred).

# Download and build

```bash
$ git clone git://github.com/julianhyde/eigenbase-xom.git
$ cd eigenbase-xom
$ mvn install
```
